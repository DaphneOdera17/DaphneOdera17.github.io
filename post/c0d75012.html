hexo 添加 aplayer 播放器

安装插件

先安装插件 hexo-tag-aplayer

修改配置

在 hexo 的根目录配置文件 _config.yml 添加


再在主题配置文件中，以 _config.butterfly.yml 中找到 aplayerInject
将 enable, per_page 改为 true

这里的 data-id 对应相应音乐软件播放列表的 id

以网易云为例子，在官网的界面打开，看链接可以找到 id
