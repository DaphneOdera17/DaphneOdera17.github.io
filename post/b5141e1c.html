Linux | 安装并配置 Ubuntu 22.04 LTS

笔者采用的是 VMware Workstation 软件。

前期准备

VMware Workstation 软件

官方下载网址
该软件似乎是需要破解的，请读者自行解决。

镜像下载

Ubuntu 官网
点击并下载镜像。
笔者用的是 22.04 版本的镜像，当然也可以选择最新版。安装步骤应该差不多。

安装系统


勾选稍后安装操作系统

版本选择 Ubuntu 64位

名称和位置根据自己偏好设置

处理器根据自己电脑情况来，后续也可以进行调整

内存默认 4GB，也可以调整

选择 网络地址转换(NAT)


个人认为磁盘大小 50GB 比较合理

磁盘文件根据给出的来就行


找到你 iso 的下载路径


设置完成后打开虚拟机

选择第一个并回车（或者等待它自动跳转）

语言自行选择，尽量选英文

选择键盘布局

选择正常安装即可

选择清空整个磁盘文件并安装系统


地区选在上海

自行配置名字和密码。

安装完成后选择重启

Please remove the installation medium, then reboot

重启的时候如果停留在一个界面显示要移除安装介质并且按下 Enter 的，直接按 Enter 就行（笔者忘记截图了）。
输入密码进入系统。
